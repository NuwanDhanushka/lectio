## 📖 About

Lectio is a simple reading app that lets you **listen to books while keeping the page visible**, with real-time text highlighting.

---

## ✨ Features

* 🔊 Text-to-speech reading
* 🎯 Highlight as you listen
* 📄 Clean, distraction-free UI
